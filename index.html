EXERCICE DE SYNTHÈSE

Identifier les compétences du poste ciblé

🎯 Objectifs pédagogiques
  • Analyser un profil métier spécifique
  • Distinguer les compétences prioritaires
  • Éliminer les compétences hors-sujet
  • Optimiser un CV pour un recruteur
🧩 Description du défi

Vous devez construire le CV d'un candidat pour un poste spécifique. Pour réussir, vous devrez faire le tri parmi 37 blocs de compétences.

⚠️ Vigilance requise : Certaines compétences sont volontairement intégrées pour tester votre capacité à identifier le cœur d'un métier.
